# Daily Discipline & Income Builder CLI

## Purpose
This project is a command-line Python application designed to help students and beginners **build daily discipline** while working toward small income-generating goals. It encourages consistency and tracks progress in a simple, interactive way.

## Features
- **Create Today Plan:** Set your main goal and 3 tasks for the day  
- **View Tasks:** Check tasks and their status (Done / Not Done)  
- **Mark Task Done:** Update tasks as completed  
- **Daily Summary & Streak:** Track completion streaks for motivation  
- **CLI-Based:** Runs entirely in terminal, easy to use on any machine  

## How to Use
1. Open `main.py` in your terminal or any Python IDE  
2. Run the program:  
   ```bash
   python main.py
